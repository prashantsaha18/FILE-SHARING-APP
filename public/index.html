File Protocols

Mount NexDrop as a Network Drive (WebDAV)

NexDrop supports the standard WebDAV protocol (RFC 4918). Access your files directly from Windows Explorer, macOS Finder, or Linux Files — just like a local USB drive. WebDAV supports file locking (LOCK/UNLOCK) and property management (PROPPATCH).

Your Personal WebDAV Endpoint http://localhost:3000/webdav
RFC 4918 Compliant File Locking (LOCK/UNLOCK) Quota Enforcement

How to Mount on Your OS

  1. Open This PC in Windows File Explorer.
  2. Click Computer in the top ribbon → Map network drive.
  3. Select a drive letter (e.g., Z:).
  4. In the Folder field, paste your WebDAV URL above.
  5. Check Reconnect at sign-in and Connect using different credentials → Finish.
  6. Enter your NexDrop username and password when prompted.
  1. Open Finder on your Mac.
  2. Press Cmd + K (or Go → Connect to Server…).
  3. Paste your WebDAV URL in the Server Address box.
  4. Click Connect, select Registered User.
  5. Enter your NexDrop username and password → Connect.
  1. Open your file manager (GNOME Files / Nautilus).
  2. Click + Other Locations in the sidebar.
  3. In the Connect to Server box, enter:
    dav://localhost:3000/webdav
  4. Click Connect, enter credentials when prompted.
